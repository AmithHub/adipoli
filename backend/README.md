# Adipoli API

Spring Boot starter backend for Adipoli.

## Planned role

- Serve drink catalog and drink detail data
- Replace the frontend mock-data service layer
- Later support recommendations, reviews, user drink history, and leaderboards

## Starter endpoints

- `GET /api/drinks`
- `GET /api/drinks/{id}`
- `GET /api/drinks/trending`
- `GET /api/drinks/top-rated`

## Notes

Maven is not installed in the current environment, so this backend structure was added but not executed here.
